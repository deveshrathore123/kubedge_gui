Welcome to Kubedge Web App Hosting Tool

Welcome to the Kubedge hosting tool! With this tool, you can easily host static websites and deploy them on Docker with minimal effort.

Step 1: Check Prerequisites

  • Ensure Docker is installed on your machine.
  • Ensure Docker Compose is installed.
  • If not, install them using apt install docker.io and apt install docker-compose on Ubuntu.

Step 2: Choose Your Website Type

Choose the type of static web app you want to host:

  • 1. HTML Only - A simple HTML-based website.
  • 2. CSS Only - A website with CSS styles.
  • 3. HTML & CSS - A fully styled website with HTML and CSS.

Step 3: Set Up Your Web Application

Provide the directory path for your static web application, including the index.html file. You will also be asked for the Docker image name, container port, and the host port.

Step 4: Dockerize and Deploy

After providing the necessary details, the tool will automatically:

  • Create a Dockerfile and docker-compose.yml in the project directory.
  • Build and deploy your app using Docker Compose.
  • Run the app and expose it on the specified host port.

Important:

Ensure your index.html is in the root of your project directory. Otherwise, the deployment will fail.

Step 5: Access Your Web App

Once deployed, you can access your website by navigating to http://localhost:8080 (or the host port you specified).
